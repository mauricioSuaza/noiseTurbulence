# noiseTurbulence
Interactive Kinect project written inn processing for commercial exhibitions.

Based on this code by the awesome Raven Kwok  

http://www.openprocessing.org/sketch/143842

- Dependencies

  Processing OpenNi.

Instructions

  Raise your hand to begin tracking.
